# SkoroMarket

Статический вариант курсового проекта без сервера.

## Как запустить

Откройте файл `index.html` двойным кликом в браузере.

## Что внутри

- `index.html` - главная страница сайта.
- `app.js` - логика каталога, регистрации, корзины, заказов и админ-панели.
- `styles.css` - оформление сайта.
- `assets/` - локальные фотографии товаров.

## Локальная база данных

Проект использует IndexedDB - встроенную базу данных браузера. В ней сохраняются:

- зарегистрированные пользователи;
- товары каталога;
- изменения из админ-панели.

Корзина и текущий пользователь дополнительно сохраняются в `localStorage`.

## Админ-панель

Админ-панель открывается кнопкой `Админ` в шапке сайта или через адрес:

```text
index.html#admin
```

В ней можно добавлять, редактировать и удалять товары. Для фото указывается локальный путь, например:

```text
assets/dj-milk.webp
```
