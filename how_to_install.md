# Инструкция по запуску двух курсовых проектов

Проекты переведены в простой статический формат: сервер запускать не нужно.

## Как открыть

1. Распакуйте нужный архив.
2. Откройте папку проекта.
3. Дважды нажмите на `index.html`.

Проект откроется в браузере как обычный локальный HTML-файл.

## Luminara

Ювелирная витрина по типу Sunlight.

Файлы:

- `index.html` - сайт;
- `app.js` - логика сайта;
- `styles.css` - стили;
- `assets/` - фотографии товаров.

Что есть на сайте:

- каталог украшений;
- категории и поиск;
- корзина;
- регистрация и вход;
- оформление демонстрационного заказа;
- админ-панель для товаров.

Админ-панель:

```text
index.html#admin
```

## Где база данных

Отдельный файл базы данных не создается. База находится внутри браузера:

- товары и пользователи хранятся в IndexedDB;
- корзина и текущая сессия хранятся в localStorage.

Это локальное хранилище на компьютере пользователя. Если открыть сайт в другом браузере, база будет отдельной.
